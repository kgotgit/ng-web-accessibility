Keyboard Navigation

↑  Up Arrow Move cursor from bottom to top
↓  Down Arrow Move cursor from top to bottom
TAB Navigate from one active element to another
Space bar Select/Unselect current focused option
CTRL + A Select/Unselect all options
HOME Moves the focus to the first option
END Moves the focus to the last option
SHIFT + ↑ Select's and keeps the one option above the current one
SHIFT + ↓ Select's and keeps the one option below the current one
