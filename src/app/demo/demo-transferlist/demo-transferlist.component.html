Keyboard Navigation

↑  Up Arrow Move focus to next option
↓  Down Arrow Move focus to previous option
TAB Navigate next active element
SHIFT + TAB Navigate next previous element
Space bar Select/Unselect current focused option
CTRL + A Select/Unselect all options
HOME Moves the focus to the first option
END Moves the focus to the last option
SHIFT + ↑ Moves toe focus to previous option and toggles the selected state
SHIFT + ↓ Moves toe focus to next option and toggles the selected state
